# Vibe Music 🎵

## Overview

**Vibe Music** is a web-based music player that allows users to browse and play songs through a simple and interactive interface. The application focuses on delivering a smooth music listening experience with essential playback controls and a clean user interface.

This project demonstrates frontend development skills including UI design, responsive layout, and interactive audio controls.

---

## Features

* 🎧 Play and pause music
* ⏭ Next and previous track navigation
* 📊 Interactive progress bar
* 🎵 Song selection from playlist
* 💻 Responsive and clean UI
* 🔊 Smooth audio playback

---

## Tech Stack

* **HTML5** – Structure of the web application
* **CSS3** – Styling and layout design
* **JavaScript** – Music player functionality and interactivity

---

## Project Structure

```
Vibe-Music
│
├── index.html        # Main webpage
├── style.css         # Styling for the interface
├── script.js         # Music player logic
├── songs/            # Audio files
└── images/           # Album art and icons
```

---

## Getting Started

1. Clone the repository

```
git clone https://github.com/yourusername/vibe-music.git
```

2. Navigate to the project folder.

3. Open **index.html** in your browser to run the application.
